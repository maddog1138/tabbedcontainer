Tabs with CSS3 & HTML5 only

This is a sample project illustrating the use of css3 and html to create simple tabbed panes. The css and html code in this project has been styled on the following codepen: Tabs with CSS3 & HTML5 only. Visit this codepen to also see some examples of content formatting within tabbed panes.

The code has been modified somewhat to update the css syntax and styling, and remove some of the internal content formatting rules. However the mechanics of the tabbed pane structure remain largely the same. The tabbed pane functionality in this example has been achieved with the help of CSS styling of radio input html tags.
